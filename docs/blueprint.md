# **App Name**: CineView

## Core Features:

- Movie Listing: Display a list of movies with titles, year, genres, and posters.
- Genre Filtering: Allow users to filter movies by genre using URL search parameters.
- Search Functionality: Enable users to search for movies by title.
- Sorting Options: Provide sorting options such as rating and year using URL search parameters.
- Movie Detail Page: Display detailed information about a movie, including description, director, cast, and related movies.
- Static Data Import: Import movie data from a static JSON file.
- Related Movies Recommendation: Suggest movies related to the one currently being viewed using a tool to filter movies of the same genres.

## Style Guidelines:

- Primary color: Dark blue (#1A202C) to establish a cinema-like environment and provide a calming visual experience.
- Background color: Very dark gray (#0A0E14), nearly black, to ensure readability of the content.
- Accent color: Electric purple (#BE4CF7) to highlight interactive elements and calls to action.
- Body text font: 'Inter', a sans-serif font, for clean readability and a modern, machined feel.
- Headline font: 'Space Grotesk', a sans-serif font, to give the headers a computerized, techy, and scientific feel.
- Use minimal, glowing icons to represent genres and interactive elements.
- Grid layout for movie listings, rounded cards, glassmorphism navbar, and clean spacing for content clarity.
